# BrainDrop

BrainDrop is a SwiftUI flashcard battle game prototype.

## Project Structure

- `BrainDrop/` - app source code (models, view models, views, assets)
- `BrainDrop.xcodeproj/` - Xcode project

## Run Locally

1. Open `BrainDrop.xcodeproj` in Xcode.
2. Select the `BrainDrop` scheme.
3. Configure Signing in `Signing & Capabilities` (your Apple Team).
4. Build and run on simulator or device.

## Notes

- Keep source files under `BrainDrop/` only.
- Do not commit build artifacts or Xcode user data.
